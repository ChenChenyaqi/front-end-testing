# 为什么要写测试

## 目标

- 认识到写测试可以早下班
- 直观的感受到写测试带给自己的好处

## 大纲

- 提高开发效率
  - 更高阶的调试手法
    - 验证局部逻辑 一步到位
- 对修改代码更加自信
  - 重构代码
  - 新增功能
- 减少重复工作
